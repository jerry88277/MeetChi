# Devlog 2026-07-15

## 線#1 落地：pre-Whisper 頻譜 Gate（開頭多人交談幻覺抑制）

承接 2026-07-10 研究（`docs/research/uat03_hallucination_analysis/`，Gate 設計已定案、
待實作進 GPU pipeline）。本次把該 Gate 從研究落地為生產程式並部署驗證。

### 實作
- **新增 `apps/backend/app/spectral_gate.py`**（純 numpy.fft，community-1 環境無 scipy）：
  - 逐窗特徵（win 1.5s / hop 0.5s）：頻譜傾斜 `10·log10(E[2–4kHz]/E[<750Hz])`、
    低頻 CPP（倒頻譜峰突起，harmonic clarity）、高頻占比 `E[2–4kHz]/E_total`。
  - `score = robust_z(tilt) + robust_z(cpp)`，每場以 **median/MAD** 自適應正規化（抗房間/麥克風漂移）。
  - **full 模式（預設）**：全檔遲滯，把持續「低分且悶」的區段遮成靜音（mask-not-delete，
    時間軸不變），交 faster-whisper VAD 跳過。**絕對高頻護欄**（hf≥0.06 一律不遮）保護真實語音；
    另有最大遮罩比例上限與最短遮罩持續護欄。備 leading 模式但實測會被錄音起始高頻瞬態誤判，故非預設。
- **整合** `offline_asr_community1.py::_transcribe_sync`：`ENABLE_SPECTRAL_GATE` 開啟時，
  進 Whisper 前 `whisperx.load_audio → apply_gate → 傳 masked numpy 給 CT2`（免重編碼），失敗回退原檔。
- **Dockerfile.gpu-service-community1** 補 `COPY app/spectral_gate.py`（否則 import 失敗、gate 靜默失效）。
- **測試** `tests/test_spectral_gate.py` 6 案全過；後端全套 240 passed（234+6），無回歸。

### 本機驗證（uat03 前 12 分，7:52=472s 為幻覺真值邊界）
- full 模式遮 **40.6%**，其中 **290s 落在 0–472s 幻覺區、僅 2s 落在真實區（472–720s）**。
- 幻覺區訊號特徵印證研究：score 負、tilt −15~−23dB、hf 0.005–0.028；真實區 score +3~+4.5、hf 升至 0.11。

### 生產 A/B（同一 chunk_000＝uat03 前 900s，僅切換 flag，deployed 程式一致）
| 區間 | Gate OFF（baseline） | Gate ON | 效果 |
|---|--:|--:|---|
| 開頭 0–472s（幻覺區） | **175 段** | **33 段** | **−81%** |
| 真實 472–900s（業務內容） | 140 段 | 144 段 | **保留（未劣化）** |

- baseline 開頭為典型會前 murmur 幻覺：`哇 / 這兩盞車 / 我來搞方啦 / 孤字「我」「你」/ 做我覺得要蘋果`。
- 真實區內容兩者皆完整連貫：`因為產業屬性的關係我們現在的水股就越來越大 / …指標7.0 / 蠻常被政府單位關注 / 所以我們今天想提出一個解方`。

### 誠實結論
- Gate 把開頭幻覺段數砍 81%，**且不損真實語音**（絕對高頻護欄有效）。
- 非 100% 消除：殘留 33 段中含少量真實會前對話與零星幻覺 loop（遮罩間隙讓 Whisper 仍能拼段）。
  屬「訊號層 Gate + 既有參數抑制」疊加後的合理殘量，較 baseline 為大幅改善。
- flag-gated，回滾＝`ENABLE_SPECTRAL_GATE=false`。
