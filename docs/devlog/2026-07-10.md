## 14:51:00 +0800 — commit `785b66a`（研究）＋本 devlog 補記
- **主線**：Whisper 幻覺 **pre-ASR 頻譜 Gate** 調查（與上方「深度報告」為兩條獨立主線；本線承接 2026-07-08「Whisper 幻覺抑制（參數型）」之後的殘留問題）。
- **動機**：參數型抑制（temperature ladder + compression/log_prob/no_speech threshold + VAD）部署後，uat03 會議前段仍殘留幻覺。使用者以人耳定位邊界在 **7:52（472s）**，並指出頻譜圖在該點前於**低頻區（<750Hz）**有明顯分界；其餘折線特徵（centroid/ZCR 等）以單一水平線難以切開幻覺/真實區。
- **第一性原理 + MECE**：幻覺區非靜音（同 -22 dBFS，故 audio_stats 判「健康」）、非白噪（flatness 較低）→ 是**低頻主導、悶、穩定、遠場 murmur，缺高頻 articulation**。判別維度拆為三軸：A 頻譜形狀（傾斜）／B 低頻諧波結構（CPP/flatness/contrast）／C 時間調變。
- **實證驗證**（本機 numpy+scipy+soundfile，前 720s，以 7:52 為真值，每特徵取最佳單一門檻的 balanced accuracy）：

  | 特徵 | Cohen's d | 單門檻 bal-acc | 方向 |
  |---|---|---|---|
  | **A1 頻譜傾斜（2–4kHz / <750Hz, dB）** | **2.39** | **90%** | 高=真實（最強單一） |
  | B1 低頻 CPP（諧波清晰度, dB） | 1.71 | 84% | 高=真實 |
  | B2 低頻(<750Hz) flatness | 0.89 | 70% | 高=糊/幻覺 |
  | B3 低頻 spectral contrast | 0.80 | 68% | — |
  | **A1+B1 多變量** | **2.54** | **92%** | 決策邊界落在 7:52 |
  | A1+B1+B2 | 2.76 | 92% | 邊際增益 |

- **關鍵結論**：判別在**頻譜「形狀＋結構」**而非音量（RMS d=0.02 無效）；使用者「<750Hz 低頻是關鍵」的觀察被證實——最強特徵正是以 <750Hz 為分母的頻譜傾斜。
- **Gate 設計（待實作）**：`score = z(spectral_tilt) + z(low_band_CPP)`；持續低分區段以**遲滯（hysteresis）+ 最短持續**平滑，於進 Whisper 前**遮罩（mask-not-delete）**為非語音；每場以 median/MAD **自適應正規化**抗房間/麥克風漂移。
- **產出**：`docs/research/uat03_hallucination_analysis/`（`uat03_axisB_validation.png` + report Axis-B 驗證章節），commit `785b66a`，已 push 雙遠端。
- **狀態**：**驗證完成、Gate 設計已定案，待實作進 GPU pipeline**（agents.md §7：GPU 必用 `cloudbuild-community1.yaml`）。實作前依現行「先驗證/解釋後動手」節奏，待使用者指示。
