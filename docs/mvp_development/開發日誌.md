# MeetChi MVP 開發日誌 (Development Log)

---

## 2026-01-30 - Client 端開發階段完成

### 🎯 桌面客戶端 (Tauri Client) 開發完成記錄

**狀態**: ✅ **已完成開發，進入維護階段**

#### 已完成功能清單

1. **即時轉錄模式 (Real-time Mode)**
   - WebSocket 即時語音辨識
   - VAD 語音活動偵測 (Silero VAD)
   - 串流式字幕顯示

2. **對齊模式 (Alignment Mode)**
   - Smith-Waterman 本地序列比對演算法
   - 字元級扁平化與段落映射
   - 多段落匹配支援
   - 全域重新同步機制 (Global Resync)
   - 低信心度 Fallback 顯示

3. **雙視窗架構**
   - 主視窗 (Overlay): 透明、穿透、永遠置頂
   - 設定視窗: 獨立標準視窗

4. **設定功能**
   - 音訊來源選擇 (系統音訊/麥克風)
   - 語言切換 (繁中/英文/日文)
   - 翻譯模式開關
   - 顯示模式切換
   - 字體大小調整
   - Initial Prompt 自訂

5. **講稿編輯器 (Script Editor)**
   - 雙窗格設計 (原文/翻譯)
   - 自動行號
   - 分隔符格式解析

6. **關鍵字校正系統**
   - `corrections.json` 配置檔
   - ASR 後處理自動替換

#### 固定版本資訊

- **框架**: Tauri v2
- **前端**: Next.js (SSG Export)
- **Rust 模組**: 
  - `cpal` - 音訊捕獲
  - `silero-vad` - 語音活動偵測
  - `tokio-tungstenite` - WebSocket

#### 後續規劃

- 維護階段：僅修復 Bug，不新增功能
- 新需求將優先在 Web 端實現
